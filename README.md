# roam-table-enhancer

`roam-table-enhancer` is an extension designed to enhance Roam Research’s native `{{table}}` / `{{[[table]]}}` tables.

While preserving Roam’s native table rendering, it adds spreadsheet-like interactivity to built-in tables, allowing direct editing of table content and supporting common row and column operations.

## Features

- Enhances Roam’s built-in `{{table}}` / `{{[[table]]}}` functionality
- Supports direct editing by clicking on cells
- Supports keyboard shortcuts: `Enter` to enter edit mode, `Tab` to move between cells, `Esc` to exit edit mode
- Automatically synchronizes cell changes back to the corresponding Roam block
- Supports right-click menus for adjusting table structure
- Supports adding/deleting rows
- Supports adding/deleting columns
- Maintains compatibility with Roam’s native table rendering and data structure as much as possible

## Use Cases

Ideal for scenarios where you need to continue using Roam’s native table syntax but want a more convenient editing experience, such as organizing research materials, maintaining simple data tables, or recording structured information.

`roam-table-enhancer` 是一个面向 Roam Research 原生 `{{table}}` / `{{[[table]]}}` 表格的增强插件。

它在保留 Roam 原生表格渲染方式的前提下，为内置表格补充了更接近电子表格的交互能力，让表格内容可以直接编辑，并支持常见的行列操作。

## 功能特性

- 增强 Roam 自带的 `{{table}}` / `{{[[table]]}}` 功能
- 支持直接点击单元格进行编辑
- 支持键盘操作：`Enter` 进入编辑、`Tab` 在单元格之间移动、`Esc` 取消编辑
- 自动将单元格修改同步回对应的 Roam block
- 支持右键菜单进行表格结构调整
- 支持新增 / 删除行
- 支持新增 / 删除列
- 尽量保持与 Roam 原生表格渲染和数据结构兼容

## 适用场景

适合需要继续使用 Roam 原生表格语法，但希望获得更方便编辑体验的场景，例如整理研究资料、维护简单数据表、记录结构化信息等。
